# schemejs
Scheme.js is a project designed to make building servers modular and fast.

## Install
```
npm i schemejs
```

## Example Server

``` javascript
eval(fs.readFileSync("./api.js").toString());
var s = new scheme(apijs);

app.post("/api", (req,res) => {
  if (req.body.store) {
    s.rag(req.body.store).then((data) => {
      res.json(data);
    });
  }
});
```
